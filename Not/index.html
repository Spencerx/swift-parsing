Structure Not 

public struct Not<Upstream: Parser>: Parser

A parser that succeeds if the given parser fails, and does not consume any input.

For example:

let uncommentedLine = Parse {
  Not { "//" }
  PrefixUpTo("\n")
}

This will check the input doesn't start with "//", and if it doesn't, it will return the whole input up to the first newline.
