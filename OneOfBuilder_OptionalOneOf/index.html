Structure One​OfBuilder.​Optional​One​Of 

public struct OptionalOneOf<Wrapped: Parser>: Parser

A parser that parses output from an optional parser.

You won't typically construct this parser directly, but instead will use standard if statements in a OneOfBuilder block to automatically build optional parsers:

let whitespace = OneOf {
  if shouldParseNewlines {
    "\r\n"
    "\r"
    "\n"
  }

  " "
  "\t"
}
